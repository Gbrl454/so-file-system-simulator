### Copiar arquivos

|     Comando     |                           Ação                            |
|:---------------:|:---------------------------------------------------------:|
|       cd        |                       Volta ao ROOT                       |
|      cd ..      |                      Volta uma pasta                      |
|    cd ~/path    |            Vai para o "path" a partir do ROOT             |
|    cd ./path    |         Vai para o "path" a partir da pasta atual         |
|    cd /path     |         Vai para o "path" a partir da pasta atual         |
| cd /path1/path2 | Cria uma fila e execulta "cd /path1" e depois "cd /path2" |
